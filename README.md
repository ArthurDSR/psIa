# Fluxo newsletter Inteli IA

## Descrição do Projeto

O **psIa** é um fluxo de automação criado no [n8n](https://n8n.io/) que envia uma **newsletter semanal** com informações sobre Inteligência Artificial na atualidade para os e-mails cadastrados em uma planilha.

## Funcionalidades

- **Newsletter semanal automatizada:** a cada semana, o fluxo n8n coleta os e-mails cadastrados na planilha e envia a newsletter com as últimas novidades sobre IA.
- **Cadastro de assinantes:** a inscrição na newsletter pode ser realizada de duas formas:
  - Pelo **frontend** disponibilizado neste repositório.
  - Diretamente na **planilha** de controle de assinantes.
- **Frontend informativo:** além de permitir a inscrição, o frontend também exibe o conteúdo das newsletters enviadas anteriormente.

## Tecnologias Utilizadas

- [n8n](https://n8n.io/) — plataforma de automação de fluxos de trabalho
- Planilha (Google Sheets) — gerenciamento da lista de assinantes
- Frontend — interface webem ionic para inscrição e visualização das newsletters
